# TIMER 3.0 Database Information

- *Full Database Name:* TIMER 3.0 (Tumor Immune Estimation Resource)
- *Version Used:* TIMER 3.0 Web Server (2025 Update)
- *Access Date(s):* 06-11-2025
- *Persistent Identifier/URL:* http://timer.cistrome.org/
- *Description of Data Used:* Used to analyze immune cell infiltration levels, immune associations, gene expression correlations, and survival relevance in LUAD using TCGA-derived datasets and multiple deconvolution algorithms.
- *Key Parameters/Filters Applied:* LUAD cancer type selection, immune infiltration estimation modules (e.g., TIMER, CIBERSORT, EPIC), gene expression correlation analysis, and survival modeling (Cox regression).
- *Licensing Information:* TIMER 3.0 is freely accessible for academic research; users should follow the citation and usage instructions provided on the TIMER website.
